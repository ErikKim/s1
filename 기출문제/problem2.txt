문제2(512M , 10개 항목에 대해 5초 제한)

육상 선수(1~200000)를 일렬로 세운다
각 육상선수에게는 실력 수치가 존재한다(-10000~10000)
각 육상선수가 자신을 포함하여 일렬로 된 팀을 구성할 수 있으며, 그 중 자신이 아닌 임의의 선수 한명을 제외할 수 있다.
각 육상선수별로 구성할수 있는 최대값의 합을 1,000,000,007로 나눈 나머지를 구하여라.
단 최대값은 음수가 될수 있으며, 각 선수별 최대값을 더할때는 절대값을 취해야 한다.

예제

1  <- 케이스 수
5  <- 선수 수
-1 <- 1번 선수부터의 실력
2
-3
2
-1

힌트
1 : {-1,2,(-3),2    } : 3
2 : {   2,(-3),2    } : 4
3 : {   2, -3 ,2    } : 1
4 : {   2,(-3),2    } : 4
5 : {   2,(-3),2, -1} : 3

|3|+|4|+|1|+|4|+|3|+=15

정답 : 15
